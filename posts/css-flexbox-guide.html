CSS Flexbox 完全指南

2026-07-06 · 前端 · #CSS #Flexbox #布局

Flexbox 是 CSS 中最常用的布局方式之一。本文整理了 Flexbox 的核心概念和常用技巧。

基本概念

使用 Flexbox 布局需要在父元素上设置 display: flex。子元素会自动成为弹性项目（flex item）。

.container {
  display: flex;
}

主轴与交叉轴

Flexbox 的核心是两条轴：

  • 主轴（main axis）：默认是水平方向（从左到右）
  • 交叉轴（cross axis）：默认是垂直方向（从上到下）

通过 flex-direction 可以改变主轴方向：

.container {
  display: flex;
  flex-direction: row;         /* 水平排列（默认） */
  flex-direction: column;      /* 垂直排列 */
  flex-direction: row-reverse; /* 水平反向 */
}

对齐方式

主轴对齐

justify-content 控制项目在主轴上的对齐：

.container {
  justify-content: flex-start;    /* 起点对齐 */
  justify-content: center;        /* 居中 */
  justify-content: space-between; /* 两端对齐 */
  justify-content: space-around;  /* 等间距 */
}

交叉轴对齐

align-items 控制项目在交叉轴上的对齐：

.container {
  align-items: stretch;    /* 拉伸填满（默认） */
  align-items: center;     /* 居中 */
  align-items: flex-start; /* 顶部对齐 */
  align-items: flex-end;   /* 底部对齐 */
}

项目属性

每个 flex item 上常用的属性：

  • flex-grow：放大比例（默认 0）
  • flex-shrink：缩小比例（默认 1）
  • flex-basis：初始大小
  • flex：简写，如 flex: 1
  • order：排列顺序（默认 0）

实战：居中布局

用 Flexbox 实现水平垂直居中最简单的方式：

.center {
  display: flex;
  justify-content: center;
  align-items: center;
}

总结

Flexbox 解决了 CSS 中大量的布局问题，掌握它是前端开发的基本功。建议在实际项目中多练习，很快就能熟练运用。
