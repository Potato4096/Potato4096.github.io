为什么写博客

写博客是一种很好的学习方式。通过输出倒逼输入，把学到的知识用自己的话整理出来，既能加深理解，也能帮助到其他人。

这个博客用什么技术

整个博客由纯 HTML、CSS 和 JavaScript 构建，没有任何框架依赖：

  • 静态 HTML 页面
  • 原生 CSS 样式（黑白极简风格）
  • 原生 JavaScript 处理搜索和动态渲染

文章数据存储在 JSON 文件中，前端通过 fetch 读取并渲染。这种方式简单直接，适合个人博客。
